# JVM Basics

## What is JVM?
The Java Virtual Machine (JVM) is a runtime environment that executes Java bytecode.

## How Java Runs

1. Write Java code (.java)
2. Compile using javac
3. Compiler creates bytecode (.class)
4. JVM executes the bytecode

Java Code → Compiler → Bytecode → JVM → Program Runs

## JVM Components

- Class Loader
- Runtime Data Area
- Execution Engine
- Garbage Collector

## Why JVM is Important

- Platform independent
- Memory management
- Security
