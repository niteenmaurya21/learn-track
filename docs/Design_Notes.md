# LearnTrack - Design Notes

## Project Overview
LearnTrack is a Java-based learning tracker that allows users to manage
courses and track learning progress.

## Design Approach
The project follows a simple modular structure separating logic and data.

## Main Components

1. Main Class
   Entry point of the application.

2. Course Manager
   Handles course creation and management.

3. Progress Tracker
   Tracks user learning progress.

## Project Structure

src/
├── Main.java
├── model/
│     └── Course.java
├── service/
│     └── ProgressTracker.java
